Introduction
This page contains demos, examples and test cases. They are only designed for desktop browsers and are not meant to be very pretty, their main purpose is to test and showcase different ways to do things. Use the sidebar on the left to navigate between examples. If you want you can also visit the desktop test home page of the project. It is a more serious attempt at creating a "normal" web page.
If you are using Firefox it can occasionally have a pretty erratic canvas performance. It is typically the slowest of all modern desktop browsers, sometimes nearly 300 times slower than Chrome.
